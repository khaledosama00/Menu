 Menu Ingredient Search (Python)

A simple Python project that displays a menu divided into categories (Pizza, Salads, Burgers) and allows the user to search for dishes using an ingredient or keyword.

 Project Idea

The user enters an ingredient or keyword (e.g., chicken, pizza)

The program searches through all menu items

Displays all dishes that contain the entered word

Shows a friendly message if no matching dishes are found

 Technologies Used

Python

Lists & Nested Lists

Loops

String methods (lower, in)

User input handling
